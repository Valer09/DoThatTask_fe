# In pratica ad ogni run di base dovrebbe buildare e scaricare tutte le dipendeze. Invece si può cachare, fintanto che le dipendenze non sono cambiate
# Quindi:

# Copia solo i file di configurazione prima
# COPY build.gradle.kts settings.gradle.kts gradle.properties ./    ← Cambia solo se modifichi le dipendenze
# COPY gradle ./gradle

# Scarica le dipendenze (questo layer viene cachato)
# RUN gradle dependencies --no-daemon || true                       ← Scarica dipendenze (CACHATO finché non cambi build.gradle.kts)

# Ora copia il resto del codice
# COPY . .                                                          ← Cambia quando modifichi il codice

# Build finale
# RUN gradle build --no-daemon -x test                              ← Solo compilazione, dipendenze già scaricate
